Ninja Tiles est un jeu de plateforme.

Comment Joueur : 

	- Pour sauter => toucher l'écran
	- Pour se déplacer de droite à gauche => pencher le téléphone dans la direction souhaitée
	- à la fin de la partie entrer votre nom et toucher "valider" pour sauvegarder votre score
	- Vous pouvez accéder à l'historique des scores depuis le menu principal en touchant le bouton "score"
